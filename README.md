# Custo do Cargo — Calculadora Interativa (Next.js + Tailwind)

Projeto gerado para o trabalho — cálculo do custo do cargo personalizado para a *Empresa do Trabalho*.

## Como usar localmente

1. Tenha Node.js >= 18 instalado.
2. No diretório do projeto, rode:
   ```bash
   npm install
   npm run dev
   ```
3. Abra http://localhost:3000

## Deploy no Vercel (recomendado)

1. Crie uma conta em [https://vercel.com](https://vercel.com) (se ainda não tiver).
2. No dashboard, clique em "New Project" → Import Git Repository (GitHub/GitLab/Bitbucket).
3. Aponte para este repositório (ou faça `git init` e envie para GitHub).
4. Vercel detecta Next.js automaticamente. Configure as variáveis se precisar.
5. Clique em Deploy — em minutos sua página estará no ar.

## Observações

- Este é um boilerplate mínimo. Os componentes de UI usados no protótipo podem ser substituídos por sua biblioteca preferida (shadcn/ui, Chakra, Material UI).
- Inclui `framer-motion` e `recharts` para animações e gráficos.
